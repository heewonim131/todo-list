# todo-list

간단한 To do List Back-end 단을 구현

개발환경
- Java 11, Spring boot 2.6.3
- Lombok, Mockito, JPA

기능
- todo 아이템 추가
- todo 아이템 조회
- todo 아이템 수정
- todo 아이템 삭제
- 전체 todo 리스트 조회
- 전체 todo 리스트 삭제
